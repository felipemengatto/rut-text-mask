# :trollface: :hankey: Welcome to rut-text-mask!

### Designed for use with text-mask plugin:

```
https://github.com/text-mask/text-mask
```

### Install Package
```
npm i -S rut-text-mask
```

### Import Plugin
```js
import RutTextMask from 'rut-text-mask';
```

### Basic Usage
```jsx
<MaskedInput
    mask={RutTextMask}
/>
```
